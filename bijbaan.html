Bijbaan

In mijn leven heb ik een aantal bijbaantjes gehad. Mijn eerste bijbaantje was in een supermarkt, de dirk van den broek in Hardinxveld-Giessendam. Ook heb ik een keer in de vakantie geholpen met het archief op een kantoor. Afgelopen zomer heb ik na mijn stage ook een vakantiebaantje aangeboden gekregen bij Unive, hier heb ik de hele zomer gewerkt.

Bij de Dirk van den broek was ik vakkenvuller, wat ik moest doen spreekt natuurlijk voor zich. Bij Unive zat ik op de debiteurenadminnistratie, hier was ik vooral bezig met het verwerken van bankafschriften en mutaties van/voor klanten verwerken. Wat heb ik geleerd wat ik binnen deze opleiding kan toepassen? Er zijn een aantal dingen die ik bij deze bijbaantjes heb geleerd die ik bij deze opleiding kan toepassen. Een van deze dingen is zoeken naar een oplossing op verschillende manieren: bij het bankafschrift verwerken waren soms een aantal problemen waar je zelf wat voor moest uitzoeken, naar mijn mening kan het goed van pas komen dat je moet blijven doorzoeken naar een oplossing. Ook heb ik geleerd hoe je in het algemeen met klanten en/of collega's moet omgaan, dit kan van pas komen tijdens projecten met klasgenoten, later op je stage en natuurlijk als je later een afspraak hebt met een klant.
